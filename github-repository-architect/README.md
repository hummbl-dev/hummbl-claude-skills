# GitHub Repository Architect Skill

**Complete GitHub repository setup with production-grade standards.**

## Description

Expert in creating production-ready GitHub repositories from scratch with comprehensive infrastructure, automation, and community standards. Provides end-to-end setup for open-source projects, enterprise codebases, and documentation sites.

## Usage

This skill is automatically triggered when working with:
- GitHub repository setup
- Repository infrastructure
- Community health files
- CI/CD automation
- GitHub Pages deployment

## Key Capabilities

- Repository initialization and configuration
- Community health files (README, LICENSE, CONTRIBUTING, etc.)
- Issue and PR templates
- CI/CD workflows (GitHub Actions)
- Documentation site deployment (GitHub Pages)
- Branch protection and security

## Installation

Add this skill to your Claude environment by placing the `github-repository-architect/` directory in your skills folder.
