# HUMMBL Base120 Mental Models Framework Skill

**Version 1.0-beta (Definitive Reference) - Complete Base120 mental models framework with all 120 models across 6 transformations.**

## Description

Comprehensive reference for the HUMMBL Base120 framework featuring 120 validated mental models across 6 transformation categories with precise codes, official names, and one-line definitions. This is the definitive, production-ready reference validated on October 16, 2025.

## Usage

This skill is automatically triggered when working with:
- Mental model reference and application
- Model selection for problem-solving
- Transformation analysis (P, IN, CO, DE, RE, SY)
- MCP server development with mental models
- Agent training and coordination
- Structured problem-solving

## Key Capabilities

- **All 120 Models:** Complete reference with official codes (P1-P20, IN1-IN20, CO1-CO20, DE1-DE20, RE1-RE20, SY1-SY20)
- **6 Transformations:** Perspective/Identity, Inversion, Composition, Decomposition, Recursion, Meta-Systems
- **One-Line Definitions:** Precise, actionable definitions for each model
- **Model Selection Guidance:** Quick reference table for choosing appropriate models
- **Validation Checklist:** Ensure correct model usage and avoid generic substitutions
- **Provenance Information:** Complete source documentation and version control

## Model Structure

Each transformation contains 20 models presented in a structured table format:
- **Code:** Transformation prefix + number (e.g., P1, IN15, CO7)
- **Official Name:** Standardized model name
- **One-Line Definition:** Precise, actionable description
