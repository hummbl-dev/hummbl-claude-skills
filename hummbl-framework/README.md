# HUMMBL Framework Skill

**Complete HUMMBL Base120 mental models reference with all 120 models across 6 transformations.**

## Description

Comprehensive reference for the HUMMBL framework including all 120 validated mental models, transformation methodologies, model selection guidance (SY19), quality standards, and application templates.

## Usage

This skill is automatically triggered when working with:
- Mental model applications
- HUMMBL framework
- Base120 models
- Transformation analysis
- Model selection
- Problem-solving with mental models

## Key Capabilities

- **All 120 Models:** Complete P, IN, CO, DE, RE, SY series
- **6 Transformations:** Perspective, Inversion, Composition, Decomposition, Recursion, Meta-Systems
- **Model Selection (SY19):** Framework for choosing right models
- **Application Templates:** Step-by-step process for each transformation
- **Quality Standards:** 9.2/10 validation criteria
- **MCP Integration:** Database schema and endpoint design

## Installation

Add this skill to your Claude environment by placing the `hummbl-framework/` directory in your skills folder.
