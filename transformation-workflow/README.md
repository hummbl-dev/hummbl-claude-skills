# Transformation Workflow Skill

**Practical application guide for HUMMBL's 6 transformations with workflows, templates, and examples.**

## Description

Step-by-step workflows for applying HUMMBL's 6 transformations (Perspective, Inversion, Composition, Decomposition, Recursion, Meta-Systems) to real-world problems. Includes when-to-use guides, combination patterns, output templates, and common pitfalls.

## Usage

This skill is automatically triggered when working with:
- Transformation application
- Problem-solving workflows
- Mental model selection
- Analysis templates
- Strategic decision-making
- Model combination patterns

## Key Capabilities

- **6 Complete Workflows:** Step-by-step for each transformation
- **When-to-Use Guides:** Indicators for transformation selection
- **Combination Patterns:** How to use multiple transformations together
- **Output Templates:** Structured formats for analysis
- **Real Examples:** Practical applications with outcomes
- **Pitfall Avoidance:** Common mistakes and solutions

## Installation

Add this skill to your Claude environment by placing the `transformation-workflow/` directory in your skills folder.
