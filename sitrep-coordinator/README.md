# SITREP Coordinator Skill

**Military-style Situation Report (SITREP) generation for multi-agent coordination.**

## Description

Expert in military-style Situation Report generation for coordinating complex projects across multiple AI agents, human operators, and distributed teams. Provides structured, actionable status updates following military communication protocols adapted for software development.

## Usage

This skill is automatically triggered when working with:
- Status reports and updates
- Multi-agent coordination
- Project handoffs
- Blocker escalation
- Team communication

## Key Capabilities

- Structured SITREP generation with standard military format
- Authorization code management for mission tracking
- Multi-agent handoff protocols
- Status tracking with color-coded indicators
- Escalation and decision support frameworks

## Installation

Add this skill to your Claude environment by placing the `sitrep-coordinator/` directory in your skills folder.
