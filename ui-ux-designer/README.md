# UI/UX Designer Skill

**Modern UI/UX design specialist for web and mobile applications.**

## Description

Expert in user interface and user experience design covering design principles, user research, interaction patterns, accessibility (WCAG 2.1), responsive design, modern frameworks (Tailwind, shadcn/ui, Material UI), prototyping, and usability testing.

## Usage

This skill is automatically triggered when working with:
- UI/UX design
- Web application interfaces
- Mobile app design
- Design systems
- Accessibility (WCAG)
- Responsive design
- Component libraries
- User experience optimization

## Key Capabilities

- **Design Principles:** Visual hierarchy, consistency, affordance, feedback, typography, color theory
- **User Research:** Personas, user journeys, pain points, usability testing, A/B testing
- **Interaction Patterns:** Navigation, forms, microinteractions, transitions, error handling
- **Accessibility:** WCAG 2.1 compliance, screen readers, keyboard navigation, color contrast
- **Modern Stack:** Tailwind CSS, shadcn/ui, Material UI, Chakra UI, Lucide icons
- **Responsive Design:** Mobile-first approach, flexible grids, touch targets, breakpoints
- **Complete Workflow:** Research → Ideation → Design → Implementation → Testing

## Installation

Add this skill to your Claude environment by placing the `ui-ux-designer/` directory in your skills folder.
